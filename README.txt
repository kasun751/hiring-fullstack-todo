# 📝 Full Stack TODO Application

A modern full-stack TODO application built using **React (TypeScript)** for the frontend and **Node.js + Express + MongoDB** for the backend.

This project demonstrates real-world full-stack development practices including clean architecture, API design, validation, and modern frontend state management.

---

## 🚀 Tech Stack

### 🖥 Frontend
- React 18+
- TypeScript
- React Query (@tanstack/react-query)
- React Hook Form
- Zod
- Axios
- Tailwind CSS
- React Hot Toast

### ⚙️ Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- express-validator
- dotenv
- cors

---

## ✨ Features

### 🧩 Core Features
- Create TODO
- View all TODOs
- Edit TODO
- Delete TODO
- Mark TODO as completed / pending

### ⚡ Frontend Features
- Modern responsive UI
- Modal-based create/edit forms
- Form validation (Zod)
- Loading & error states
- Toast notifications
- Optimistic UI updates (React Query)

### 🛡 Backend Features
- RESTful API design
- Request validation middleware
- Clean layered architecture
- MongoDB persistence
- Centralized error handling
